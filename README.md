# ProjectTwitter – Clone di Twitter

Questo progetto è un'applicazione clone semplificata di Twitter, sviluppata in Java utilizzando il framework **Spring Boot**.  
L'applicazione include funzionalità base di gestione utenti, autenticazione, tweet e interfaccia web con Thymeleaf.

---

## 🚀 Tecnologie utilizzate

- Java 11
- Spring Boot 3.2.1
- Spring Security
- Spring Data JPA
- Spring Boot Thymeleaf
- Maven
- MySQL
- Servlet API

---

## ✨ Funzionalità implementate

- Registrazione e login utente (autenticazione con Spring Security)
- Creazione, visualizzazione e gestione dei tweet
- Interfaccia web responsive con Thymeleaf
- Persistenza dei dati con MySQL e JPA
- Struttura pulita MVC (Model–View–Controller)

---

## 🛠️ Requisiti per l'esecuzione

- Java 11+
- Maven
- MySQL in esecuzione (database: `twitter`)
- IDE consigliato: IntelliJ IDEA o Eclipse
